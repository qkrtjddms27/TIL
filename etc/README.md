### 앱

앱 마켓 사용가능

알람, 주소록, 카메라, GPS등의 스마트폰 기능을 사용할 수 있음.

서비스 실행이 편함.

스마트폰에 최적화 되어있어 속도가 빠름.

UI사용성이 스마트폰에 최적화 되어 있음.

------

마켓에 등록하기 위해 개발자 등록을 해야함.

사용자가 앱을 다운 받아야하는 불편함

IOS, AOS모두 개발 해야함.

앱이 수정되면 마켓에 새로 배포하여야 한다.

사용자가 사용하는 디바이스에 OS에 따라 버전을 관리해야 한다.
